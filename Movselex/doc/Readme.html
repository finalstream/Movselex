Movselex¶

手軽な動画の管理と多彩な条件でセレクションができる動画セレクター
Support OS : Microsoft Windows 10 / 8 / 7 / Vista .Net Framework 4.5 is required.

ダウンロードしていただきありがとうございます。

MovselexはのコンセプトはMovselecと同じです。最新技術を取り入れ、UIを刷新しています。

インストーラはありません。レジストリも使用しません。
インストールおよびアップデート方法についてはhow to install or updateをご覧ください。

また、再生に使用するプレイヤーとして現在のところMedia Player Classicのみ対応しています。
起動後に設定画面よりインストール先のパスを設定する必要があります。
お持ちでない方は以下からダウンロードしてください。
Media Player Classic - Home Cinema(English)
http://mpc-hc.sourceforge.net/

はじめはD&Dで動画ファイルを登録していただければよいです。
以降は一番多く登録されているパスについては自動更新して自動で登録されます。（将来的には監視フォルダを登録できるようにする予定です）
また、動画ファイルを更新することはありませんのでご安心ください。

Movselecのデータベースと互換がありますのでMovselexのdatabaseフォルダにMovselecのdbフォルダのファイルを入れてから起動してください。
現在、マニュアルはありませんのでご不明な点はtwitter等でご質問いただければと思います。

最新の開発状況はこちらをご覧ください。チケットを発行していただくことでご要望やバグ報告も受け付けています。
https://github.com/finalstream/Movselex

ソフトの概要¶

  • ハードディスクに動画があちこちに。
  • 手間をかけずに動画を管理したい。
  • 動画のファイル名がバラバラ。
  • あの動画どこに保存したっけな。
  • こういう条件で動画をランダムに再生できればなー。

上記のようなお悩みはすべてMovselexで解決できます。

Movselexは手軽な動画の管理と多彩な条件によって動画をセレクションできる動画セレクターです。
再生に使用できるプレイヤーは現在、MPC-HCのみですが、今後プラグイン化し、いろいろなプレイヤーに対応する予定です。

まだまだ開発中であるため今後にご期待ください。
